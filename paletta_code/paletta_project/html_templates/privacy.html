Privacy

1. Data Collection

We collect and store the following information to operate and improve the Paletta platform:

  • Basic user data such as name, email address, and user role (e.g., teacher, student).
  • Metadata related to uploaded files (e.g., upload time, file type, storage usage).
  • Optional licensing data, including licensee details for publicly shared content.

2. Consent and Releases

It is the responsibility of users to ensure that all individuals identifiable in uploaded content have signed appropriate release forms or have equivalent institutional permissions.

Paletta is developing tools that allow users to upload or link signed consent forms to specific files for improved record-keeping and compliance.

3. Use of AI Tools

Paletta may employ AI technologies to analyse and tag uploaded media to enhance search functions and user experience:

  • No biometric or facial recognition data is used unless explicitly permitted and legally justified.
  • All AI-generated tags are used strictly for internal purposes and are never sold or shared with third parties.

4. Data Storage and Security

We take data security seriously and implement the following measures:

  • All user data and uploaded content are securely stored with role-based access control.
  • Content is hosted on GDPR-compliant servers located in [insert server region, e.g., the European Economic Area].
  • We apply encryption, regular security audits, and strong access protocols to ensure data integrity and prevent unauthorised access.
